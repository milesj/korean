The suffix `으러` can be conjugated with a verb (the stem) in order to express the purpose of "in
order to". For example, I went to the store _in order to_ buy a gift.

## Guidelines

- The suffix `으러` is used when the previous syllable ends in a consonant.
- The suffix `러` is used when the previous syllable ends in a vowel.

## Examples

I'm going to the store _in order to_ buy a gift.

```
저는 선물을 사러 가게에 거요.
```
